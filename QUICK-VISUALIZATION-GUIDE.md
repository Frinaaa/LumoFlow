## Troubleshooting

### No visualization appears?
- Make sure you have code in the editor
- Check that the file is a JavaScript file
- Try clicking Play button again

### Voice not working?
- Check the sound toggle (should be purple)
- Make sure browser allows audio
- Try refreshing the page

### Visualization too fast?
- The system automatically waits for voice
- Each step pauses for 800ms
- Voice speed is set to 0.65x (slow)

### Want to see a specific step?
- Use the progress bar
- Click Play/Pause to control flow
- Watch the step counter
